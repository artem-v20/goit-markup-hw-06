# Web Studio

It's a basic website for a studio, featuring a main page with sections: **Hero**, **Features**, **Team**, **Portfolio**, and a footer with a subscription form.

## 📂 Structure

- `index.html` — main page of the site
- `css/styles.css` — core styles
- `images/` — graphics and icons for sections

## 🛠 Technologies Used

- **HTML5** — semantic markup
- **CSS3** — styling and responsiveness
- **Modern Normalize** — cross‑browser style normalization
- **Google Fonts (Raleway, Roboto)** — custom fonts

## ✨ Features

- Navigation menu and responsive header
- Button to open the mobile menu
- Content sections with SVG icons
- Subscription form in the footer
- Modal window with a contact form

## 🎯 Purpose

This project helped me:

- strengthen my knowledge of HTML and CSS
- practice building responsive layouts
- understand the basics of working with forms and SVG icons
